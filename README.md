# Project_Disney_Alura
Projeto inspirado na Disney+ com apoio da imersão Alura, conta com seleção de perfis, tema claro/escuro, cards de conteúdo e página de exibição. Só o inicio do projeto, e contou com bastante ajuda de IA (um dos focos da imersão), e por ter apenas experiência com CSS e HTML, tive que pedir bastante ajuda para o JScript.
