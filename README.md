# Project_Saturn
Projeto inspirado na Disney+ com apoio da imersão Alura, conta com seleção de perfis, tema claro/escuro, cards de conteúdo e página de exibição. Só o inicio do projeto, e contou com bastante ajuda de IA (um dos focos da imersão), e por ter apenas experiência com CSS e HTML, a parte do JScript foi bastante "robotizada", com muitas oportunidades de mudança para o futuro, conforme for avançando em meus estudos.

🔹 Próximas melhorias planejadas:

Criar identidade visual própria (nome, logo e conteúdo)

Implementar interações com JavaScript (modais, carrossel, player)

Integração com API de filmes (TMDB)

Sistema de busca dinâmica

Implementar Dark/Light Mode

Melhorar animações e experiência do usuário

Refinar responsividade
